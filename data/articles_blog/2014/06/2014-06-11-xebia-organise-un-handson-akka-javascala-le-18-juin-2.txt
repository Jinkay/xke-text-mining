2014-06-11-xebia-organise-un-handson-akka-javascala-le-18-juin-2|CraftEvents|   Xebia vous propsose une soirée Tech’Events autour du framework Akka le mercredi 18 juin. Xavier Bucchiotty et Vincent Spiewak vous proposent de découvrir la programmation asynchrone autour d’un jeu sur le thème de la gestion de trafic aéroportuaire.  Vous découvrirez le modèle acteur, comment l’utiliser et quelles sont les contraintes de conception qu’il amène dans nos applications. Nous aborderons en fin de séance les sujets de persistence et d’Event Sourcing. L’atelier peut se dérouler en Java ou en Scala. L’inscription se fait sur EventBrite. Les pré-requis sont :  un environnement de développement Java (JDK), Typesafe Activator ≥ 1.2.2  Nous utiliserons Typesafe Activator lors de cet atelier. Cet outil nous permettra de dérouler le tutorial. Il sert aussi d’éditeur en ligne. Le repository de code est disponible sur Github Vous pouvez d’ores et déjà récupérer le code et l’installer avant l’atelier. Quelques mises à jour mineures du code sont cependant à prévoir jusqu’à la date de l’atelier.
