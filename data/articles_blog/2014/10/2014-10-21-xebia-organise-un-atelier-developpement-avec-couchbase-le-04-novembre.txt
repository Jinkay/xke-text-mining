2014-10-21-xebia-organise-un-atelier-developpement-avec-couchbase-le-04-novembre|DataEvents|  Si vous vous intéressez un peu au NoSQL, vous avez forcément dû voir passer le nom de Couchbase ces derniers temps. Xebia organise dans ses locaux un hands-on pour apprendre à manipuler cette base de données orientée à la fois clé/valeur et documents. Vous aurez l’occasion d’aborder des problématiques de modélisation, d’asynchronisme et de performance. Vous pourrez administrer un ou plusieurs nœuds dans le cloud afin de découvrir la web console Couchbase. et vous pourrez essayer la version 2.0 du SDK Java (assez de son prédécesseur).   En deux heures, vous découvrirez comment est conçu Couchbase, les nouveautés de la 3.0 et comment utiliser la version 2.0 du SDK Java (assez différente de son prédécesseur) avec en prime des utilisations de lambda de Java 8. Nous aurons aussi la chance d’accueillir Frédéric Berqué, de chez Couchbase, qui nous introduira cette session avec une présentation traitant d’architecture et de uses cases. Cet atelier est donc destiné à tous les développeurs avec un minimum de connaissances en Java et désireux de découvrir cette base de données ou les nouvelles versions sorties récemment. Cette session sera animée par Mathieu Breton, Jean-Eudes Couignoux, Rodolphe Bung et Antoine Michaud. Pour les inscriptions c’est ici. Audience : Développeur Java désireux de découvrir Couchbase Plan de l’atelier :  Présentation architecture et uses cases Hands’on  Date : mardi 04 novembre 2014 à 19h Lieu : Xebia156, boulevard Haussmann à ParisPremière à gauche, 7ème étage Pré-requis : Un pc avec un JDK 1.7 minimum, mais version 1.8 conseillée, Maven, votre IDE préféré et Git. Ou, dans le pire des cas, vous trouverez bien un binôme sur place… Prix : Gratuit, restauration comprise
