2015-02-25-deux-hackathons-ethereum-les-5-et-20-mars-pour-decouvrir-les-applications-decentralisees|EventsFront|   Vous connaissez sûrement bitcoin, la monnaie cryptographique. Elle se base sur une technologie encore mal connue: la blockchain. Si on creuse le sujet, on s’aperçoit que bitcoin n’est qu’un use case parmi de nombreux autres.  Quel est le programme ? Chez Xebia, parmi ces usecases, ce sont les applications décentralisées qui nous passionnent. Imaginez construire des programmes décentralisés (sans serveur) qui peuvent s’exécuter entre deux utilisateurs (ou machines). On parle alors de refaire des versions décentralisées de eBay, Uber, Spotify. Ethereum est un acteur majeur de ces nouvelles technologies. L’ouverture du système Ethereum sur lequel pourront s’exécuter ces applications décentralisées est prévue pour fin mars (peut être le 20 mars). Nous vous proposons de venir coder vos premières applications   Dapps   dans nos locaux le 5 mars avec des Xebians pour vous accompagner. En fin de soirée nous présenterons les résultats de chaque équipe. Une seconde soirée est prévue le 20 mars avec comme objectif de sortir les premières Dapps sur Ethereum.
