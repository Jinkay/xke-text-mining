2015-05-18-tech-event-initiation-au-machine-learning-avec-spark-le-4-juin|Data|  Savoir faire du Machine Learning, notamment dans un contexte Big Data, est une compétence de plus en plus recherchée de nos jours. Pourtant cette discipline est loin d’être la plus simple à acquérir. Xebia vous propose de vous initier progressivement au Machine Learning et au framework Spark. Nous utiliserons les outils les plus récents des Data Scientists au travers de la librairie MLlib de Apache Spark, framework reconnu dans le domaine du Machine Learning et du traitement de gros volume dans l’éco-système Java. Nous utiliserons Scala tout au long du hands’on (à un niveau très basique, n’ayez pas peur si vous débutez !). Vous serez guidé par un tutoriel (en plus des animateurs) tout au long des exercices. Ce Tech-Event sera animé par Yoann Benoit, Alban Phélip et Mathieu Breton. Pour les inscriptions, c’est ici. Programme Cet atelier, qui a déjà été présenté lors des conférences Devoxx France et Mix-IT, vous permettra de découvrir théoriquement et par la pratique les sujets suivants:  Compréhension globale du Machine Learning et de ses enjeux Introduction générale à Spark Présentation de la librairie MLlib pour le Machine Learning sous Spark      Utilisation de MLlib en pratique  Pour de l’apprentissage non-supervisé (algorithme du K-Means) Pour de l’apprentissage supervisé (arbres de décisions) Utilisation de techniques avancées de classification (RandomForest, Grid-Search)    Pré-requis Ce hands’on est ouvert à tous, sans connaissance préalable sur le Machine Learning particulière. Le projet vous sera fourni sous forme d’une Machine Virtuelle comprenant tous les outils nécessaires pour cette session. Les postes que vous amenez doivent donc avoir les configurations suivantes :  2Go de RAM au minimum (4Go conseillé) 15Go de disque dur disponible Virtualbox ou VMware La machine doit pouvoir supporter  une image ‘invité’ VMware 64 bits. Si la machine dispose d’une version 64 bits  de Windows ou Mac OS X avec un processeur Core DUO 2 ou au delà, aucun test n’est requis. Le cas échéant, VMware fournit un outil permettant de vérifier la compatibilité : https://my.vmware.com/web/vmware/details/processor_check_5_5_dt/dCpiQGhkYmRAZQ== Le support VT-x virtualization doit aussi être activé dans le BIOS  Informations pratiques Date : le 4 Juin 2015 à 19h Lieu : Xebia 156 boulevard Haussmann à Paris, 75008 ParisImmeuble A – 7ème étage Prix : Gratuit, buffet c agnard compris
